Introduction

Discrete Mathematics is a branch of mathematics that deals with distinct and countable structures.

It is widely used in computer science, especially in algorithms, data structures, and logic design.

Unlike continuous mathematics, discrete mathematics focuses on integers, graphs, and logical statements.

It provides the foundation for problem-solving and computational thinking.

Discrete = finite or countable values
Logic

Logic is the study of reasoning and truth values.

It is used to determine whether a statement is true or false.

In computer science, logic is used in decision-making processes such as conditions in programming.

Logical expressions are built using operators like AND, OR, and NOT.

  • Proposition: A statement that is either true or false
  • Truth table: A table showing all possible values
  • Logical operators: AND (∧), OR (∨), NOT (¬)
p ∧ q (AND operator) p ∨ q (OR operator) ¬p (NOT operator)
Sets

A set is a well-defined collection of distinct objects.

Elements inside a set are enclosed within curly brackets.

Sets are used in databases, probability, and programming.

There are different types of sets such as finite, infinite, and empty sets.

  • {1, 2, 3} → finite set
  • {a, b, c} → set of characters
  • ∅ → empty set
A ∪ B (Union) A ∩ B (Intersection)

Union combines elements from two sets, while intersection finds common elements.

Relations

A relation defines a connection between elements of two sets.

It is a subset of the Cartesian product of two sets.

Relations are important in database design and graph theory.

They help describe how elements are linked together.

  • Reflexive: Every element is related to itself
  • Symmetric: If (a,b) then (b,a)
  • Transitive: If (a,b) and (b,c), then (a,c)
(a, b) ∈ R R ⊆ A × B
Functions

A function is a special type of relation where each input has exactly one output.

Functions are widely used in both mathematics and programming.

They help represent relationships between variables.

Functions must follow the rule of uniqueness (one input → one output).

  • Domain: Set of input values
  • Range: Set of output values
  • Mapping: Relationship between input and output
f(x) = x² f(2) = 4
