# Personal Portfolio Website (FreeCodeCamp Individual Assignment)

This is a **Personal Portfolio Website** developed as part of the **FreeCodeCamp Responsive Web Design Certification**. The purpose of this task is to demonstrate fundamental web development skills using **HTML and CSS**, while organizing and showcasing several completed assignments.

The portfolio acts as a central hub that links to four major projects:
- Tribute Page
- Survey Form
- Product Landing Page
- Technical Documentation Page

Each task was built following FreeCodeCamp requirements, focusing on:
- Responsive design
- Clean layout structure
- Proper use of semantic HTML
- CSS styling and positioning

This assignment also reflects an understanding of how to structure a real-world website and present work professionally.
